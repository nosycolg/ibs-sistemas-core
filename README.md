# IBS-SISTEMAS-CORE

### Environment Variables

Environment Variables can be set on .env file.

### How to Install?

```bash
nvm use
npm install
```

### How to run?

```bash
make up
```

### How to run tests?

```bash
nvm use
npm run tests
```

### How to build it?

```bash
nvm use
npm run build
```
